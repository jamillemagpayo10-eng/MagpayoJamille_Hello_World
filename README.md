# Description
This project was created for Activity 3 in the IS 312 course. It shows how to set up a basic website using Python by starting a local server and creating a simple web page. When the program runs, it displays a "Jamille F. Magpayo, Hello World!" message.

# Framework Used
Flask - A small and simple Python tool used to build websites quickly without needing complex settings.

# How to Run the Application Locally
1. Open PyCharm.
2. Open your terminal and run: pip install flask
3. Create a new Python project named MagpayoJamille_Hello_World.
4. Under the project directory, create a new Python file named app.py.
5. Copy the code from MagpayoJamille_Hello_World/app.py and paste it into your local app.py.
6. Run the program.
7. Open a web browser and navigate to the local server URL (e.g.,http://127.0.0.1:8000).
8. The message "Jamille F. Magpayo, Hello World!" should appear on the page.
